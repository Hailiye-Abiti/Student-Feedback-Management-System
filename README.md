#Wachemo University

A secure, web-based platform designed to bridge the gap between students and university administration. This system allows students to provide structured feedback on instructors, dormitory services, and cafeteria facilities to improve campus life.

🚀 Features
Role-Based Access: Separate dashboards for Students (submission) and Admins (reporting).

Dynamic Forms: Questions change automatically based on the selected category (Dorm, Cafe, or Instructor).

Security: Passwords are protected using Bcrypt Hashing and inputs are sanitized to prevent SQL injection.

Responsive Design: Works perfectly on Mobile, Tablets, and Desktops.

Printable Reports: Admins can generate physical reports with one click.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Flexbox ), JavaScript .

Backend: PHP .

Database: MySQL.

Authentication: PHP Session Management.
